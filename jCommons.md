# 日语常见祠

| Type   | Japanese                                                    | Chinese |
|------------|-----------------------------------------------------------|------------|
|          |    彼ら（かれら）　　　 |    他们   |
|          |   彼女（かのじょ） 　　　 |   她    |
|          |    男（おとこ）　　　 |   男性    |
|          |    女（おんな）　　　 |    女性   |
|          |    名字（みょうじ）　　　 |  姓     |
|          |    名前（なまえ）　　　 |    名字   |
|          |    やる　　　　　　　 |  简单理解：する的口语化用词，做的意思     |




日本人姓氏举例：石原里美（いしはら　さとみ）、石原（いしはら）は名字（みょうじ）です、
里美（さとみ）は名前（なまえ）です。
